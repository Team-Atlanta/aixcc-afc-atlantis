# Write your own Interprocedural Dataflow Analysis

### Dependencies
=== "Maven"
    ```maven
    <dependency>
        <groupId>org.soot-oss</groupId>
        <artifactId>sootup.analysis</artifactId>
        <version></version>
    </dependency>
    ```
=== "Gradle"
    ```groovy
        compile "org.soot-oss:sootup.analysis:{{ git_latest_release }}"
    ```
